# 一. 项目研究目的
本项目旨在解决隐私保护学习(PPML)中，密态神经网络因非线性操作产生的性能瓶颈，特别是无法充分利用 GPU 并行计算能力的问题。现有基于同态加密(FHE)或安全多方计算(MPC)的方案在处理非线性函数时，或依赖高次多项式逼近引入巨额计算深度与自举开销，或需进行大量通信交互，导致推理延迟超过明文计算，难以实用。本项研究目的是，通过设计 GPU 友好的低深度，高精度算法，提升密态神经网络中非线性层的执行效率，推动高安全、高性能的隐私计算在实际场景中的落地应用。
# 二. 项目研究内容
本项目以CryptoFace端到端加密人脸识别系统作为研究背景，聚焦其在全同态加密(FHE)环境下运行时面临的性能瓶颈——密态神经网络中非线性操作近似运算。在CryptoFace中，为实现从加密图像输入到匹配结果输出的全流程隐私保护，必须在密态下执行卷积、激活、归一化等操作。然而同态加密仅天然支持加法和乘法运算，对于ReLU、GELU、L2归一化等非线性函数，必须通过低深度、可密态实现的数学近似来替代，否则会导致乘法深度过大、自举开销剧增，使推理延迟难以接受。本项目的研究内容是在CryptoFace框架内，设计更高效的密态非线性操作近似算法，并探索利用GPU（CUDA/Tensor Core）并行计算对其进行提速。
# 三. 国内外研究现状和发展动态
当前，密态神经网络推理主要沿三条技术路径发展：全同态加密（FHE）、安全多方计算（MPC）和可信执行环境（TEE）。FHE方案（如CryptoFace）虽能提供端到端加密，但非线性操作需通过高次多项式逼近，导致计算深度大、自举开销高，严重制约性能，且现有实现大多基于CPU，难以利用GPU。MPC方案（如基于秘密分享的方法）通过多方协同计算来保护隐私，其线性层计算高效，但非线性层（如比较、除法）通常依赖通信密集的混淆电路（GC）或复杂协议，同样面临延迟挑战，且现有的GPU加速方案不完善，对Softmax、GELU等复杂算子支持有限。TEE方案性能接近明文，但受限于硬件可用性和内存容量，并面临侧信道攻击风险。
# 四. 研究路线及解决的主要问题
本项目总体遵循前期学习、论文复现、尝试调优的研究路线，从基础知识与工具掌握出发，到理解并复现CryptoFace加密面部系统的密态神经网络的非线性操作近似运算，再到针对性能瓶颈进行创新优化与GPU加速尝试，逐步深入并产出可验证的成果。
本项目将主要解决以下问题：前期学习阶段，主要解决缺乏相关密码学、并行计算等背景知识的问题，通过建立系统的知识框架为后续项目的推进打牢基础；论文复现阶段，解决不了解近似算法在整体系统中的位置与作用的问题，通过完整流程对系统整体进行了解与掌握；尝试调优阶段，解决原CryptoFace非线性近似低效率的问题，尝试在保持原有精度的前提下提升算法的效率；解决密态非线性运算无法利用GPU并行运算的问题，尝试通过CUDA和Tensor Core实现并行优化。
# 五. 项目创新及特色
本项目创新与特色在于：采用“前期学习—论文复现—尝试调优”的路线，从CryptoFace的实际性能瓶颈切入，提出算法优化与CUDA/Tensor Core并行化结合的协同优化思路，兼顾算法效率与硬件加速；逐步分层推进，确保可实现性，为密态神经网络实用化提供轻量化研究范式。
# 六. 项目综述
本项目基于密态神经网络推理中的性能瓶颈——非线性操作近似运算，以CryptoFace端到端加密人脸识别系统为载体，针对其密态推理中非线性近似效率低、难以利用GPU并行的问题展开研究。项目采用“前期学习—论文复现—尝试调优”的渐进路线，先通过系统学习建立知识框架，再复现CryptoFace流程以明确近似算法的作用与瓶颈，最后尝试CUDA与Tensor Core并行化优化。创新在于将算法优化与硬件加速协同，最终形成可验证的代码。
# 七. 项目实施方案
本项目将围绕研究路线分阶段制定可执行计划，确保各阶段目标与待解决问题的落地。团队由三人组成，明确分工：一人负责前期学习与论文复现中的理论吸收、CryptoFace核心模块理解与代码复现；一人负责尝试调优中的算法改进与GPU加速实现；另一人负责数据准备、实验记录与成果整理。
在时间安排上：前三个月为前期学习阶段，集中进行文献调研与理论学习，内容包括同态加密（CKKS）、CryptoFace论文阅读，非线性近似运算原理，GPU体系结构与CUDA编程入门，完成学习笔记与知识框架梳理，解决背景知识缺乏的问题。第四到第六个月为论文复现阶段，下载并分析CryptoFace的代码与公开数据集（如LFW），在CPU上复现其密态推理流程，重点实现非线性近似模块（如L2归一化的二次多项式近似），完成加密→推理→解密的端到端测试，测量各模块耗时与识别精度，理解近似算法在整体系统中的位置与作用。第七到第九个月为尝试调优阶段，将核心计算用CUDA重写，利用GPU并行处理批量数据，优化内存访问（合并访存、共享内存），尝试用Tensor Core批量计算，验证输出正确性并测量GPU相对CPU的加速比，解决原非线性近似效率低与无法利用GPU并行的问题。第十到第十二个月进行全面性能测试与对比分析，将调优后的模块替换回CryptoFace框架进行端到端测试，整理数据并撰写结题报告与成果展示材料。
# 八. 项目预期成果
本项目预期形成一套完整可验证的研究成果，体现从知识积累到系统复现再到创新优化的全过程。首先，将产出系统的学习笔记与知识框架文档，清晰呈现同态加密、密态神经网络非线性近似、GPU并行计算等核心知识点的内在联系，解决前期学习阶段背景知识缺乏的问题。其次，将提供CryptoFace核心模块的CPU复现代码，包含端到端加密推理流程的实现、性能与相关数据，能够直观展示近似算法在整体系统中的作用与位置，达成论文复现阶段的目标。第四，将尝试实现基于CUDA与Tensor Core的GPU提速，在批量密态数据上验证并行化效果，测得GPU相对CPU的加速比，为密态非线性运算利用GPU并行提供可行路径与实验依据。
