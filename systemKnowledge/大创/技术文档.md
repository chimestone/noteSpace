# 大创项目技术文档

## 一、项目整体认知

### 你在做什么（一句话）
让神经网络在"看不见数据"的情况下完成人脸识别。

### 系统三层架构

```
第一层：加密层
用户的人脸图像 → 加密 → 变成一堆密文数字
（从这一刻起，任何人包括服务器都看不见原始图像）

第二层：密态推理层（项目核心）
密文 → 神经网络各层计算 → 还是密文
问题出在这里：神经网络里有 ReLU、L2归一化这些非线性操作
FHE 只认加法和乘法，所以必须用多项式去"假装"这些操作

第三层：解密层
密文结果 → 解密 → 得到匹配分数 → 判断是不是同一个人
```

### 项目核心问题链

```
人脸识别 → 需要隐私保护 → 用同态加密(FHE) →
FHE只支持加减乘 → 神经网络有非线性函数(ReLU/GELU/L2) →
必须用多项式近似替代 → 次数越高越慢 →
现有方案太慢且不能用GPU → 本项目要解决这个
```

### 你要做的事
优化第二层里"假装"非线性操作的过程，让它更快、更准，并且能用 GPU 跑。

## 二、三阶段任务拆解

### 第一阶段：前期学习（前3个月）

**目标：** 建立知识框架，能用自己的话解释项目在做什么

#### 需要掌握的核心概念

| 概念 | 只需理解到什么程度 |
|------|------------------|
| 同态加密 / CKKS | 知道它支持加法和乘法，乘法有深度限制，自举开销大 |
| 乘法深度 | 多项式次数越高，深度越大，开销越大 |
| 非线性近似 | 用低次多项式逼近 ReLU/GELU/L2，次数越低越快但误差越大 |
| 切比雪夫多项式 | 最常用的近似方法，记住这个名字 |
| GPU / CUDA | SM、线程块、共享内存、Tensor Core 的基本概念 |

#### 学习顺序建议
1. 搜索"CKKS同态加密 知乎"，看一篇中文科普，重点记住**乘法深度**
2. 用 NumPy 手写一个 ReLU 的切比雪夫多项式逼近，感受误差
3. 安装 TenSEAL，跑通官方示例
4. 安装 CUDA Toolkit，跑通向量加法示例
5. 阅读 CryptoFace 论文摘要 + 引言 + 结论

## 三、快速上手：先跑通代码再说原理

> 在开题报告前，先跑出一个直观结果，建立感知

### 推荐最快路径
1. 安装 TenSEAL：`pip install tenseal`
2. 跑通官方加密向量加法示例
3. 用 TenSEAL 实现一个加密状态下的简单多项式计算（如 x² + 2x + 1）
4. 对比明文计算结果，感受近似误差

### 示例：切比雪夫近似 ReLU（明文验证）
```python
import numpy as np
import matplotlib.pyplot as plt

# 用多项式近似 ReLU（示意，非精确切比雪夫）
def poly_relu(x):
    return 0.5 * x + 0.25 * x**2

x = np.linspace(-2, 2, 100)
true_relu = np.maximum(0, x)
approx_relu = poly_relu(x)

print("最大误差:", np.max(np.abs(true_relu - approx_relu)))

plt.plot(x, true_relu, label="真实 ReLU")
plt.plot(x, approx_relu, label="多项式近似", linestyle="--")
plt.legend()
plt.title("ReLU 多项式近似误差")
plt.show()
```

### 示例：TenSEAL 加密计算入门
```python
import tenseal as ts
import numpy as np

# 创建 CKKS 上下文
context = ts.context(
    ts.SCHEME_TYPE.CKKS,
    poly_modulus_degree=8192,
    coeff_mod_bit_sizes=[60, 40, 40, 60]
)
context.generate_galois_keys()
context.global_scale = 2**40

# 加密一个向量
plain = [1.0, 2.0, 3.0, 4.0]
encrypted = ts.ckks_vector(context, plain)

# 密态下做多项式计算：x^2 + 2x + 1
result = encrypted ** 2 + encrypted * 2 + 1

# 解密并对比
decrypted = result.decrypt()
expected = [x**2 + 2*x + 1 for x in plain]

print("解密结果:", [round(v, 4) for v in decrypted])
print("期望结果:", expected)
print("误差:", np.max(np.abs(np.array(decrypted) - np.array(expected))))
```

## 六、关键术语速查

| 术语                | 一句话解释                     |
| ----------------- | ------------------------- |
| FHE               | 全同态加密，允许在密文上直接计算          |
| CKKS              | FHE的一种方案，支持浮点数近似计算        |
| 乘法深度              | 电路中串行乘法的层数，越深开销越大         |
| 自举（Bootstrapping） | 刷新密文噪声以继续计算，开销极大          |
| 切比雪夫近似            | 用多项式逼近非线性函数的常用方法          |
| Tensor Core       | NVIDIA GPU 中专为矩阵乘法设计的计算单元 |
| NTT               | 数论变换，FHE 内部多项式乘法的核心操作     |
| LFW               | 人脸识别标准测试数据集               |
