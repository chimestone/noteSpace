## 第一阶段：前期学习（前3个月）

### 1. 同态加密基础

你的项目核心是 CKKS 方案（CryptoFace 使用的就是它）。

学习顺序建议：

- 先理解**对称加密 → 公钥加密 → 同态加密**的概念演进，不需要深入数学，先建立直觉
    
- 重点理解 CKKS 的三个核心特性：
    
    - 支持浮点数近似计算
        
    - 支持加法和乘法（但乘法有"深度"限制）
        
    - 需要"自举（Bootstrapping）"来刷新计算深度
        
- 推荐入门资料：
    
    - [CKKS 原论文](https://eprint.iacr.org/2016/421.pdf)（先看摘要和引言即可）
        
    - Microsoft SEAL 库的官方文档和示例（有 Python 绑定，上手快）
        
    - YouTube 搜索 "CKKS homomorphic encryption explained"
        

关键概念你必须搞清楚：

- 明文空间 vs 密文空间
    
- 乘法深度（Multiplicative Depth）
    
- 噪声预算
    
- 自举开销为什么大
    

### 2. 密态神经网络非线性近似

这是你项目的核心问题所在。

为什么非线性是瓶颈：

- FHE 只支持加法和乘法，ReLU、GELU、L2归一化这些函数**无法直接计算**
    
- 必须用**多项式**来逼近，但多项式次数越高，乘法深度越大，开销越大
    

你需要学习的内容：

- 切比雪夫多项式逼近（Chebyshev Approximation）——这是最常用的方法
    
- 最小二乘多项式拟合
    
- 理解"低深度"和"高精度"之间的 trade-off
    

推荐资料：

- CryptoFace 原论文（你项目的核心文献，必读）
    
- [HEAR](https://eprint.iacr.org/2021/1265.pdf)、[CryptoNets](https://proceedings.mlr.press/v48/gilad-bachrach16.pdf) 等密态推理论文，了解业界如何处理非线性
    

### 3. GPU 与 CUDA 基础

学习顺序：

- 先理解 GPU 架构：SM（流多处理器）、线程块、线程束（Warp）、共享内存
    
- 再学 CUDA 编程模型：kernel 函数、grid/block/thread 层次、内存层次
    
- 最后了解 Tensor Core：专为矩阵乘法设计，FP16/BF16 精度
    

推荐资料：

- NVIDIA 官方 [CUDA C++ Programming Guide](https://docs.nvidia.com/cuda/cuda-c-programming-guide/)（看前几章）
    
- [CUDA by Example](https://developer.nvidia.com/cuda-example)（入门书，有中文版）
    
- PyTorch 的 
    
    ```plaintext
    torch.cuda
    ```
    
     和自定义 CUDA 扩展文档

## 第三阶段：尝试调优（第7-9个月）

### GPU 加速的切入点

不是所有操作都适合 GPU 加速，你需要优先关注：

- 批量多项式求值（可以并行化）
    
- 矩阵乘法（卷积层，Tensor Core 友好）
    
- NTT（数论变换，FHE 内部核心操作，已有 GPU 实现参考）

## 立即可以开始做的事

按这个顺序启动：

1. 安装 TenSEAL，跑通官方示例（1-2天）
    
2. 阅读 CryptoFace 论文，标注你看不懂的部分（1周）
    
3. 用 NumPy 手写一个 ReLU 的切比雪夫多项式逼近，感受误差（2-3天）
    
4. 安装 CUDA Toolkit，跑通向量加法的 CUDA 示例（1-2天）
    

这四件事做完，你对整个项目的技术栈就有了基本感知，后续学习会快很多。
