最一开始，我对于代码的粗浅理解在于各种IDE里面的文本格式，例如：
```c
# include <stdio.h>
int main(void)
{
	printf("hello world!");
	return 0;
}
```
这是一段最简单的代码，效果是把"hello world"这个字符串打印出来。
接下来，我们将以它为引子，介绍计算机是如何从代码文本转化为可执行文件的。
简单来说，就是
1. 预处理（Pre-processor）
2. 编译（Compiler）
3. 汇编（Assembler）
4. 链接（Linker）
这四个过程。
# **预处理**
首先，我们的这段代码包含对头文件的引用：
```c
# include <stdio.h>
```
在预处理阶段，我们的代码文件以hello.c格式输入，预处理器将头文件的相干内容直接插入文件生成hello.i文件，进行下一步。
# **编译**
通过一系列的分析与处理，将上一阶段得到的hello.i文件处理为hello.s文件，并进行下一步。
# **汇编**
汇编器根据指令集将**汇编程序**hello.s翻译为机器指令，并将这些机器指令根据一定的规则进行打包，得到可重定位文件hello.o。
# **链接**
在hello中，我们调用了头文件当中的printf函数，被包含在一个称为printf.o的已经编译好的文件。链接阶段就是将这些文件打包生成可执行文件的过程。
