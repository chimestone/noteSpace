# 概述
这是一种特殊的加密方式，旨在对原数据未知的前提下，允许对加密后的数据直接进行数学运算（如加法，乘法等），运算结果解密后，与对原数据进行相同运算的结果一致。
# 为何适用
**神经网络**的本质就是大量的加法和乘法运算，因此理论上能够在加密前提下运行神经网络。

# 全同态加密

**全同态加密**是一种允许在加密数据上进行运算的加密技术，计算结果仍然是加密的，以保护数据隐私。
