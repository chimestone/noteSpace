# 概述
密态神经网络是一个结合了**密码学**和**人工智能**的技术概念，简言之，它是指**在数据处于加密状态下，直接进行神经网络计算的技术**。
核心目的就是保护数据隐私和*模型知识产权*。
# 实现方式
实现方式主要依赖**同态加密**技术。
[[同态加密|>>指路]]
当然还涉及其他技术，如安全多方计算，混淆电路等，此处暂不涉及。
# 总结
密态神经网络=神经网络+同态加密
