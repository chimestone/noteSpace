# 概述
神经网络是一种仿照人脑结构创造的运算模型。
我们通过数学概念模拟人脑的活动过程。由大量的人工神经元（节点）通过连接组成；每一个连接都有其权值，表示其重要程度。我们要通过调整权重，让该模型能够学习到规律，进而完成特定的功能。
# 基本结构
有三层组成：
1. 输入层
2. 隐藏层：对输入内容进行处理
3. 输出层
# 工作原理
数据在神经网络中是单向流动的，即从输入到处理再到输出，这个过程叫做前向传播。
前向传播会有以下两个过程：
1. 加权求和：每个神经元会接受来自上一层所有的神经元的输入，每个输入乘以对应的连接权重，然后求和。
2. 激活函数：求和的值会被送入一个激活函数，这个函数的作用是引入非线性。非线性使得我们的模型得以处理更加复杂的问题。
