# 补码加法运算
由于机器内数据以补码格式存在，因此设计加法运算即为：
$$
[X + Y]_补 = [X]_补 + [Y]_补
$$
## 公式
对于整数：
$$
[X]_补 = \{^{X \space 0 \leq X < 2^n}_{2^{n+1} + X \space -2^n \leq X < 0} \space \mod 2^{n+1}
$$
对于小数：
$$
[X]_补 = \{ ^{X \space 0 \leq X < 1} _{2 + X \space -1 \leq X < 0}\space \mod 2
$$
数学块修正：
$[X]_{\text{补}} = \begin{cases} X, & 0 \le X < 1 \\ 2+X, & -1 \le X < 0 \end{cases} \mod 2$
## 减法的拓展
$$
[X - Y]_补 = [X]_补 + [-Y]_补
$$
需注意：
$$
[Y - Y]_补 = [Y]_补 + [-Y]_补 = 0
$$
